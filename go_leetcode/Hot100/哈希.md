# 两数之和

用哈希表存储的是目标结果和每个数字的差，之后再遍历数组，看看有没有和哈希表中匹配的值

```go
func twoSum(nums []int, target int) []int {
    //创建哈希表
    results := make(map[int]int) 
    //得出计算结果，判断是否存在与哈希表中，并且存储在哈希表中
    for i , value := range nums{
        result := target - value
        //检测存在性
        if index,exists := results[result];exists{
            return []int{index,i}
        }
        //结果写入哈希表
        //results[result] = i 出错
        results[value] = i 
    }
    //返回空值
    return nil
}
```

# 字母异位词分组

学一下字符串排列就行，这个是最简单的

1. **生成键**：对每个字符串进行排序，排序后的字符串作为键。因为字母异位词排序后会得到相同的字符串。
2. **哈希表存储**：使用哈希表，键是排序后的字符串，值是所有原始字符串的列表。
3. **返回结果**：遍历哈希表的值，将所有的分组收集到一个列表中返回。

```go
import (
	"sort"
	"strings"
)

func groupAnagrams(strs []string) [][]string {
	groups := make(map[string][]string)//一个字符串匹配多个字符串
	for _, s := range strs {
		key := sortString(s)//将顺序值作为键
		groups[key] = append(groups[key], s)//将字符串作为值进行存储
	}
	result := make([][]string, 0, len(groups))
	for _, v := range groups {
		result = append(result, v)
	}
	return result
}

func sortString(s string) string {
	chars := strings.Split(s, "")
	sort.Strings(chars)//按照字符串顺序进行排列
	return strings.Join(chars, "")
}
```

# 最长连续序列

使用哈希集合来存储所有数字，以便快速查找某个数字是否存在

```go
func longestConsecutive(nums []int) int {
    //哈希表存储 数字是否存在
    numSet := make(map[int]bool)
    for _, num := range nums {
        numSet[num] = true
    }
    
    maxLength := 0
    for num := range numSet {
        if !numSet[num-1] {//在这个数字之前不存在数字了（这个数字是起始的）
            currentNum := num
            currentLength := 1
            //查看下一个数字是否存在
            for numSet[currentNum+1] {
                currentNum++
                currentLength++
            }//跳出循环说明数字中断了，更新最大长度
            if currentLength > maxLength {
                maxLength = currentLength
            }
        }
    }
    return maxLength
}
```
