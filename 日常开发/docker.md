# 前置内容

## 基础概念

1. 【容器】docker容器可以与传统虚拟机进行类比，不同的地方在于docker容器之间共用一套操作系统内核
2. 【镜像】镜像是容器的模板，可以类比为软件的安装包，镜像与容器的关系就像是模具与产品的关系
3. 【镜像仓库】docker的官方镜像仓库是docker Hub

## 安装

【安装指令地址】[get.docker.com](https://get.docker.com/) 执行1 4 两条命令

1. `curl -fsSL https://get.docker.com -o install-docker.sh`
2. `sudo sh install-docker.sh`
