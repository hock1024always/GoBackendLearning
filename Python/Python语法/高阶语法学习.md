# 学习优先级

基于外包项目的管理系统以及音视频功能

### 技术应用场景对应表

| 技术模块   | Web管理系统应用场景      | 一对一视频系统应用场景      |
| :--------- | :----------------------- | :-------------------------- |
| subprocess | 系统维护任务             | 调用FFmpeg进行视频转码/处理 |
| asyncio    | 异步API接口              | 实时视频流传输、信令服务器  |
| threading  | 后台任务处理             | 并行音视频编码              |
| logging    | 操作审计、错误追踪       | 通话质量监控、连接问题诊断  |
| datetime   | 记录创建时间、操作时间戳 | 通话计时、时间同步          |
| os/sys     | 文件管理、系统配置读取   | 设备检测、资源监控          |
| re         | 表单验证、路由解析       | SDP协议解析、信令格式验证   |
| queue      | 任务队列管理             | 视频帧缓冲、数据包排序      |

### 额外建议学习的技术（列表中未包含但关键）

- **Web框架**：FastAPI（推荐，内置异步支持）或Django
- **数据库操作**：SQLAlchemy或Django ORM
- **WebRTC**：一对一视频通信的核心技术（JavaScript+Python）
- **WebSocket**：实时通信必备（可通过aiohttp或Django Channels实现）
- **前端技术**：HTML/CSS/JavaScript + WebRTC API
